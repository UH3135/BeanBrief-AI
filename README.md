# 📘 웹서버 컴퓨팅 AD 프로젝트: 질문 게시판

## 🔍 개요
이 프로젝트는 Django 웹 프레임워크를 기반으로 한 **질문 게시판 시스템**입니다. 사용자는 질문을 등록하고, 수정 및 삭제할 수 있으며, 댓글을 달고 파일도 업로드할 수 있습니다. 또한, 교과서의 스크롤 기능과 Markdown기능을 추가했습니다.
